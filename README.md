## 安装步骤

### 前提条件

- Node.js v14.0.0 或更高版本
- npm v6.0.0 或更高版本

### 安装命令

```bash
# 克隆项目
git clone https://github.com/your-username/your-project.git

# 进入项目目录
cd your-project

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 配置说明

### 环境变量

创建 `.env` 文件并添加以下配置：

```env
# 开发环境配置
NODE_ENV=development
PORT=3000
API_BASE_URL=http://localhost:8080/api
```
